# XatuSchoolApp

## 简介


掌上西工是西安工业大学Android客户端，包含课程查询、成绩查询、课程表、学习情况分析、评教等功能模块。该项目由  合作开发完成。 

西安工业大学官网：http://www.xatu.cn/

## 体验
掌上西工已上架各大应用商店，直接百度搜索“掌上西工”即可。

## 依赖
